### Part I
La part I consisteix en quatre algorismes d'ordenació diferents, on entra per paràmetre una sèrie dodecafònica aleatòria o no. Aquesta sempre acabarà en l'escala cromàtica.  
Els algorismes triats són:
- Bubble Sort
- Bucket Sort
- Insertion Sort
- Merge Sort
